## Getting Started

Required packages:
- npm install react-bootstrap bootstrap
- npm install -g --force nodemon (sometimes the force is not needed)
- npm install express
- npm install cors
- npm install dotenv

Required APIs:
- [Weatherbits](https://www.weatherbit.io/)
- [TMDB](https://www.themoviedb.org/)
